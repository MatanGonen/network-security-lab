# \# Network Security Home Lab — Palo Alto / EVE-NG# # !\[PAN-OS](https://img.shields.io/badge/PAN--OS-10.1-blue)# !\[EVE-NG](https://img.shields.io/badge/EVE--NG-Community-orange)# !\[Wireshark](https://img.shields.io/badge/Wireshark-4.x-lightblue)# !\[Python](https://img.shields.io/badge/Python-3.10-green)# # \## Overview# Full-stack virtual network security lab built in EVE-NG on VMware Workstation.# Covers firewall policy, site-to-site IPsec VPN, SSL Forward Proxy,# and application-layer traffic analysis.# # \## Lab Topology# Linux\_LAN (192.168.10.10)# |# Switch\_HQ (VLAN 10/20)# |# PA\_HQ (10.0.0.2 / 192.168.10.1 / 192.168.20.1)# |# R\_ISP (10.0.0.1 / 10.0.0.5)# |# PA\_BRANCH (10.0.0.6 / 192.168.30.1)# |# Linux\_BRANCH (192.168.30.10)# DMZ: Linux\_DMZ (192.168.20.10) behind PA\_HQ# VPN: tunnel.1 — 10.10.10.1/30 <-> 10.10.10.2/30# # \## Tech Stack# \- \*\*Firewall:\*\* Palo Alto PAN-OS 10.1 (PA\_HQ + PA\_BRANCH)# \- \*\*Routing:\*\* Cisco vIOS (R\_ISP) — Static + OSPF# \- \*\*Endpoints:\*\* Alpine Linux (LAN, DMZ, BRANCH)# \- \*\*Platform:\*\* EVE-NG Community on VMware Workstation# \- \*\*Analysis:\*\* Wireshark, Python 3 + Netmiko# # \## Lab Modules# # | Module | Topics Covered |# |--------|----------------|# | Week 1-2 | Lab setup, VLANs, Security Zones, OSPF, NAT |# | Week 3 | Packet flow, SNAT/DNAT, FW policy, session analysis |# | Week 4 | Site-to-site IPsec VPN — full Golden Scenario |# | Week 5 | Wireshark deep dive, TCP performance metrics |# | Week 6 | HTTP/TLS analysis, SSL Forward Proxy decryption |# | Week 7 | Network automation with Python + Netmiko |# # \## Golden Scenario — VPN Troubleshooting Order# Phase 1 (IKE) -> Phase 2 (IPsec) -> Routing -> NAT -> FW Policy -> Packet Capture# # Application layer add-on:# TCP Handshake -> TLS Handshake -> HTTP Request -> App-ID -> FW Policy# # \## Repository Structure# configs/          # PA\_HQ, PA\_BRANCH, Switch\_HQ, R\_ISP configurations# failure-library/  # 40+ documented failure scenarios with root cause + fix# scripts/          # Python/Netmiko automation scripts# captures/         # Wireshark .pcap files# docs/             # Topology diagrams, flow maps# # \## Key Commands Reference# # \*\*VPN:\*\*# show vpn ike-sa# show vpn ipsec-sa# show vpn flow tunnel-id <id># # \*\*SSL Inspection:\*\*# show counter global filter aspect decrypt delta yes# # \*\*Session analysis:\*\*# show session all filter destination 192.168.20.10# show log traffic direction equal forward dst in 192.168.20.10
